3D Flood Simulation of Mannheim

Visualizing urban flood risk using QGIS, elevation models, and 3D rendering tools.

Project Overview

This project models a 3D flood scenario in Mannheim’s city center using QGIS. Digital elevation models (DEM), building footprints, and OpenStreetMap data were used to simulate inundation and visualize flood-prone areas under low elevation flood conditions.

Tools & Technologies

  • QGIS 3.44
  • QuickOSM & QGIS2ThreeJS plugins
  • DTM & DSM (Digital Terrain/Surface Models)
  • Raster Calculator, Zonal Statistics, Hillshade tools
  • OpenStreetMap buildings (GeoPackage)

Workflow Breakdown

  1. Base Map & Data Collection: Extracted building footprints and base maps via QuickOSM.
  2. Elevation Data: Processed DTM & DSM, filled NoData gaps for continuous elevation surface.
  3. Flood Simulation: Used Raster Calculator to simulate flood at ≤ 5m elevation: ("dtm_voidfilled@1" <= 5)
  4. Terrain Analysis: Slope raster created to detect barriers like dykes or hills.
  5. Building Heights: Applied Zonal Statistics on DSM to get average height per building footprint.
  6. 3D Visualization: Extruded buildings and flood polygons using QGIS2ThreeJS.

Flood Impact Statistics

  • Total buildings (GeoPackage): 11,720
  • Flooded buildings (≤ 1m): 7,935
  • Flood risk exposure: ~67.7% of buildings affected
