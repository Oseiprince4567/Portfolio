EV Charging Stations Activity Dashboard

The EV Charging Stations Activity Dashboard is a real-time, data-driven dashboard application that visualizes the spatial and temporal dynamics of electric vehicle (EV) charging stations across Hamburg using SensorThingsAPI. It integrates open IoT data from the Hamburg SensorThings API with automated data ingestion, cloud database synchronization, and interactive analytics built in Streamlit.

Rationale

  • Demonstrate real-time data integration using open urban IoT streams.
  • Provide decision-support tools for EV infrastructure planning and energy management.
  • Showcase a serverless automation pipeline (GitHub Actions ‚Üí Neon PostgreSQL ‚Üí Streamlit Cloud).
  • Support research on urban sustainability, smart mobility, and renewable-energy transitions.

Technical Stack

  • Frontend & Visualization: Streamlit, Plotly, Folium
  • Data Handling: Pandas, NumPy
  • Database: Neon PostgreSQL (cloud-hosted)
  • Backend Automation: GitHub Actions (cron job every 6 hours)
  • APIs: Hamburg SensorThings API (IoT)

Deployment Workflow

  1. Streamlit Cloud hosts the dashboard (evhamapp.py).
  2. GitHub Actions runs fillin.py every 6 hours to fetch new IoT data and update the Neon Database.
  3. Neon Database serves as persistent storage for analytics and visualizations.
